### 简要说明
```python
# 快速安装
> npm i docsify-cli -g
> docsify init ./yanghuizhi     # 项目初始化
> docsify serve yanghuihi       # 项目运行
```
PS：如果换了电脑，需要重新随便初始化一个项目，即可正常运行

